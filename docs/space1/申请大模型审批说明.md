# 申请大模型审批说明

如果是新建独立场景，请先与 guandi 沟通清楚需求后，使用通用审批进行发起审批申请使用。

其中：

- **标题**请写：`【业务线】-【业务场景名】申请接入【大模型名称】`
  - 业务线和业务场景请替换成自己的，大模型名称要写全，如 deepseek-v3，hunyuan-t120250403，doubao-lite-128k-240828
- **内容**请写：
  - 需要厂商：火山云/阿里云/腾讯云
  - 使用场景：简要说明申请的使用场景
  - TPM：预计的 TPM 峰值大小
- **节点审批人**请选择：
  - 一级审批：上级主管
  - 二级审批：企业效率组
